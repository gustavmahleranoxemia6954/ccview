ccview helps you manage your work with Claude Code. This desktop tool lets you see your installed agents, skills, and commands in one place. You can use it to track what you have built and monitor how your tools interact with your files. 

## 🛠️ What this tool does

Claude Code allows you to build custom skills and deploy agents to help with coding tasks. As your collection of tools grows, it becomes hard to track what is currently active. This application solves that problem by providing a clear dashboard for your local environment.

You can view:
- A list of all active agents on your machine.
- Every skill you have installed for use with Claude Code.
- A library of commands ready for execution.
- Status updates for your current projects.

The software presents this data in a simple window. You do not need to use the command line to check your current setup. Everything stays organized so you can focus on building your code.

## 🧭 Using the interface

The main screen is divided into sections. Use the navigation sidebar on the left to move between different data sets.

### Viewing your agents
Click the "Agents" tab to see all agents currently stored on your machine. You will see the name of each agent, the date of creation, and a short note on what the agent does. If an agent is running, the status light will turn green.

### Managing your skills
The "Skills" tab shows every module you have added to Claude Code. You can sort these by name or by the date you added them. This list helps you remove old skills that you no longer need. Use the search bar at the top of this tab to find a specific skill by name.

### Checking your commands
The "Commands" tab lists every command available to your agents. This is useful when you need to remember the syntax for a specific tool. Each command displays a description of its function.

## ⚙️ Settings and preferences

Access the settings menu by clicking the gear icon in the bottom corner of the window. 

- **Automatic refresh:** The app checks for new agents and skills every few minutes. You can toggle this setting to save resources.
- **Theme:** Choose between a light and dark interface. The app defaults to the system setting of your computer.
- **Data paths:** If you moved your Claude Code files to a custom folder, you can update the path here so the app knows where to scan for new add-ons.

## 🛡️ Privacy and data

We value your privacy. The ccview application operates locally on your machine. We do not send your agent data, skill names, or code snippets to an external server. The app reads the configuration files on your computer to display information. It does not alter your project files, modify your settings, or execute code on your behalf. All interactions stay contained within your physical hardware. 

If you choose to use the update feature, the app only connects to the GitHub repository to check if a newer version of the installer exists. 

## 🆘 Troubleshooting common issues

If the application does not show your data, check these common items:

- **Check the path:** Ensure the app knows the directory for your Claude Code configuration. You can check this in the Settings menu.
- **Restart the app:** Sometimes a simple restart clears connectivity errors between the app and your local file system.
- **Permission errors:** Run the app as an administrator if your Claude Code files reside in a protected system folder.
- **Reinstall:** If the app behaves in an unexpected way, download the latest installer again from the link provided above and run it to overwrite the old files. 

If you continue to have trouble, you can open an issue on the GitHub repository page. Provide the version number found in the "About" tab of the settings menu and a description of what you see on your screen. This helps others identify if they have similar issues.
